# 🎓 学生成绩管理系统 (Student Score Management System)

这是一个基于 C++ 开发的控制台应用程序，用于管理学生信息和成绩。系统包含完整的用户登录注册机制、权限分级（管理员/普通用户），以及对学生数据的增删改查、排序和统计分析功能。

## ✨ 主要功能 (Features)

### 🔐 用户与安全系统
*   **账户管理**：支持用户注册、登录、找回密码（通过安全问题）。
*   **安全机制**：
    *   密码输入掩码显示（显示为 `*`）。
    *   密码复杂度检测（需包含数字和字母）。
    *   验证码机制（注册时）。
    *   登录防爆破：连续输错密码自动冻结账号 10 秒。
    *   登录/退出动画效果。

### 📚 学生信息管理
*   **数据录入**：支持学号、姓名、系别、班级及三门课程（数学、英语、计算机）成绩录入。
*   **数据查询**：
    *   按学号精确查找。
    *   按姓名模糊查找。
    *   按单科成绩查询。
*   **数据修改与删除**：管理员可修改任意学生信息或删除记录。
*   **数据持久化**：所有数据自动保存至 TXT 文件 (`student_data.txt`, `user_date.txt`)。

### 📊 数据分析与排序
*   **多种排序**：支持按总分、数学、英语、计算机成绩进行排序（支持全校、按系别、按班级筛选）。
*   **统计分析**：
    *   计算各科平均分、最高分、最低分。
    *   统计优秀率（>=90分）和及格率（>=60分）。

### 👥 权限分级 (Role-Based Access)
系统分为两种角色模式：
1.  **管理员系统 (Admin)**：拥有所有权限（增删改查、统计、排序）。
2.  **学生系统 (User)**：仅拥有查询、统计、排序权限（只读）。

## 🚀 快速开始 (Getting Started)

###  初始登录说明
系统采用 **双层验证** 逻辑：

1.  **第一层：个人账号登录**
    *   首次使用请选择 `1. 注册新用户`。
    *   注册成功后，使用注册的账号密码登录。

2.  **第二层：系统权限认证** (硬编码在源码中)
    登录个人账号后，需选择进入的子系统：
    *   **管理员系统** 密码：`admin2007`
    *   **学生系统** 密码：`user0402`

## 📂 文件结构 (File Structure)

*   `main.cpp`: 程序的源代码文件。
*   `student_data.txt`: 存储学生信息的数据库文件（自动生成）。
*   `user_date.txt`: 存储注册用户账户信息的数据库文件（自动生成）。

## 📖 操作演示 (Usage Guide)

### 界面概览
程序使用了 `system("cls")` 和 `system("color ...")` 来清屏和改变控制台颜色，提供更好的交互体验。

### 常用操作键
*   菜单选择通常使用数字键 `1-9`。
*   部分确认操作使用 `Y/N`。
*   按任意键继续（Pause）。

### 示例流程
1.  启动程序 -> 欢迎动画。
2.  选择 `1` 注册账号 -> 输入用户名、验证码、符合复杂度的密码、安全问题及答案。
3.  选择 `2` 登录账号 -> 输入刚才注册的信息。
4.  进入主循环 -> 选择 `1. 管理员系统`。
5.  输入管理员口令 `admin2007`。
6.  进入管理员菜单 -> 选择 `1` 录入学生信息 -> 输入学号 `101`, 姓名 `张三`, 成绩等。
7.  选择 `7` 成绩排序 -> 查看排名。
8.  选择 `0` 退出 -> 退出动画。
